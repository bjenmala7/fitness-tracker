# 💪 Fitness Tracker Website ## Your 69kg → 50kg Journey A complete, free, online fitness tracker for managing your weight loss journey. **Features:** - ✅ Calorie tracking (eat, burn, deficit) - ✅ Weight logging with progress - ✅ Meal logging (preset + custom foods) - ✅ Workout tracking - ✅ Everything saves locally (no login needed) - ✅ Works offline --- ## How to Deploy (Free) ### Option 1: GitHub Pages (Recommended) **Easiest - takes 2 minutes** 1. Go to https://github.com/new 2. Name it `fitness-tracker` 3. Upload `index.html` to the repo 4. Go to Settings → Pages → Source: main branch 5. Your site is live at `https://yourusername.github.io/fitness-tracker` ### Option 2: Netlify (Also Free) 1. Go to https://netlify.com 2. Drag & drop `index.html` 3. Your site is live instantly 4. Can add custom domain ### Option 3: Vercel (Also Free) 1. Go to https://vercel.com 2. Upload `index.html` 3. Instant deployment ### Option 4: Just open locally - Save `index.html` on your laptop - Double-click it - Works completely offline - All data saved in browser --- ## How to Use 1. **Log weight** → Tab: Weight → Enter your weight 2. **Log meals** → Tab: Meals → Pick food or add custom 3. **Check workouts** → Tab: Workouts → Tick when done 4. **View calories** → Tab: Calories → See your daily balance Everything syncs automatically and saves to your browser. --- ## Your Plan - **Current:** 69 kg - **Goal:** 50 kg (19 weeks) - **Daily deficit:** 500 cal - **Expected loss:** 0.5 kg/week ### Workouts - Morning Yoga: 6:30–7:30 AM (180 cal) - Intense Workout: 5:30–6:30 PM (350 cal) - Total burn: ~530 cal/day ### Nutrition - Target intake: 1600 cal/day - Burn: 530 cal/day - Net deficit: 500 cal = 0.5 kg/week loss --- ## File Structure ``` fitness-tracker/ └── index.html (single file - everything inside) ``` That's it! One file. No server needed. --- ## Notes - Data saves locally in your browser (browser cache/localStorage) - Clear browser cache = data gone (be careful!) - Works on mobile & desktop - No internet needed after first load --- ## Need Help? If you want to customize: - Colors: Look for `#667eea` in the CSS - Calorie target: Change `calorieTarget = 1600` - Workouts: Modify `workoutCalories` object - Foods: Add more to `foodDatabase` --- **Happy tracking! 💪**
